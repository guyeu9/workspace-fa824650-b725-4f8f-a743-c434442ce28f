- [ ] 解析 `列列任务更新.txt`，提取所有场景分类和具体任务步骤。

- [ ] **视觉锚点**: 加入对男性特征、肌肉线条、白袜勒痕等细节描写。

- [ ] **支线跳转**: 优化步骤间的逻辑，增加失败/意外触发的紧急跳转分支。

## 3. JSON 实现
- [ ] 实现主菜单 (`main_hub`)。

## 4. 验证与交付
