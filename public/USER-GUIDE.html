🚀 快速开始

第一次游戏

  1. 打开游戏页面
    • 访问游戏网址
    • 你将看到欢迎界面
  2. 开始游戏
    • 点击"开始游戏"按钮
    • 游戏会自动加载示例故事
    • 显示起始房间信息
  3. 开始探索
    • 阅读房间描述
    • 使用指令或点击按钮进行互动
    • 享受你的文字冒险之旅！

📖 基础操作

输入指令

你可以通过两种方式与游戏互动：

🎮 快捷按钮

点击界面上的快捷按钮，这是最简单的方式：

  • 👁️ 观察 - 查看当前房间
  • 📦 物品 - 查看房间内物品
  • 🎒 背包 - 查看背包物品
  • ❓ 帮助 - 显示帮助信息

⌨️ 文本指令

在输入框中输入指令，按回车执行：

  • LOOK 或 观察 - 查看当前房间
  • GO 北 或 去北方 - 向北移动
  • TAKE 钥匙 或 拿取钥匙 - 拾取钥匙
  • USE 钥匙 或 使用钥匙 - 使用钥匙

移动指令

在不同房间之间移动：

GO [方向]      # 英文
去 [方向]        # 中文

# 支持的方向：
北/南/东/西/上/下
north/south/east/west/up/down

物品操作

TAKE [物品名]   # 拾取物品
USE [物品名]     # 使用物品
DROP [物品名]    # 丢弃物品

角色互动

TALK TO [角色名]    # 与角色对话
与 [角色名] 对话     # 中文版本

🔧 高级功能

快捷操作按钮

游戏界面提供了多个快捷按钮：

  • 👁️ 观察 - 重新查看当前房间
  • 📦 物品 - 查看当前房间内的所有物品
  • 🎒 背包 - 查看你携带的所有物品
  • 💾 保存 - 保存当前游戏进度
  • 📂 读取 - 读取之前保存的游戏
  • 🗑️ 清除 - 清除对话历史
  • ❓ 帮助 - 显示帮助信息

动态选择按钮

游戏会根据当前情境自动生成相关操作按钮：

  • 移动选项 - 显示可用的出口方向
  • 拾取选项 - 显示可拾取的物品
  • 使用选项 - 显示背包中可使用的物品
  • 对话选项 - 显示可与角色对话的主题

💡 专业提示

动态按钮会根据游戏状态实时更新，始终显示最相关的操作选项。这让你可以专注于故事体验，而不需要记住复杂的指令。

📁 JSON导入导出

导出游戏故事

  1. 点击"导出JSON"按钮
  2. 浏览器会自动下载当前故事的JSON文件
  3. 文件名格式：text-engine-story-YYYY-MM-DD.json

导入自定义故事

  1. 准备JSON格式的故事文件
  2. 点击"导入JSON"按钮
  3. 选择你的JSON文件
  4. 系统会验证并自动加载你的故事

📚 故事格式参考

如果你需要创建自己的故事，可以：

  • 下载示例JSON文件作为模板
  • 查看JSON格式说明
  • 阅读故事创作指南

❓ 常见问题

Q: 如何开始游戏？

A: 打开游戏页面，点击"开始游戏"按钮即可开始冒险。

Q: 指令不区分大小写吗？

A: 是的，所有指令都不区分大小写。你可以输入 "LOOK"、"look" 或 "Look"。

Q: 如何保存游戏进度？

A: 点击"保存"按钮，游戏会自动保存当前进度到浏览器本地存储。

Q: 可以创建自己的故事吗？

A: 当然可以！使用"导出JSON"功能下载示例文件，修改后用"导入JSON"加载你的故事。

Q: 支持中文指令吗？

A: 支持大部分常用中文指令，如"观察"、"去北方"、"拿取钥匙"等。

Q: 游戏无法加载怎么办？

A: 尝试刷新页面，或检查网络连接。如果问题持续，请联系技术支持。

Q: 如何报告bug或提出建议？

A: 点击页面右上角的"反馈"按钮，可以提交问题和建议。

Q: 支持哪些浏览器？

A: 支持所有现代浏览器，包括Chrome、Firefox、Safari、Edge等。
